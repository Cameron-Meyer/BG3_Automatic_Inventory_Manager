Executes the provided Filter against the provided params. Any exceptions will be logged, swallowed, and whatever the value of the winners table was at exception time will be returned

Parameters:

  • filter the single filter being evaluated, such as a Compare or TargetFilter
  • modifiers associated with the ItemFilter
  • customItemFilterFields any fields that aren't FILTERS or MODIFIERS that were found on the ItemFilter
  • eligiblePartyMembers GUIDSTRING[]
  • targetsWithAmountWon table
  • inventoryHolder CHARACTER
  • item GUIDSTRING
  • root GUIDSTRING

Returns:

    table winners The survivors that were eligible to receive the item, or the original survivors table if none were eligible
