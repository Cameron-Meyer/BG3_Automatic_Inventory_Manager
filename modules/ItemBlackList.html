Functions

AddEntriesToBlackList (modUUID, blacklistedItems, blacklistedRoots, blacklistedTags)
Add new items or rootTemplates to the blacklist - duplicate entries will be ignored

Parameters:

  • modUUID any
  • blacklistedItems nil or list
  • blacklistedRoots nil or list
  • blacklistedTags nil or list

Returns:

    boolean if there weren't any problems with adding the entries (will be true even if no tables were provided or all entries provided already existed)
IsItemOrTemplateInBlacklist (item, rootTemplate)
Checks to see if the item or rootTemplate is in the user-provided blacklist.

Parameters:

  • item GUIDSTRING optional. If provided, will also check against the blacklisted Tags
  • rootTemplate GUIDSTRING optional

Returns:

    boolean true if the item or rootTemplate is in the blacklist
IsContainerInBlacklist (item)
Checks the given item to see if it's a container and in the dedicated blacklist - if it isn't, will recursively check its DirectInventoryOwner

Parameters:

  • item GUIDSTRING

Returns:

    rootTemplate GUIDSTRING that was found in the blacklist
