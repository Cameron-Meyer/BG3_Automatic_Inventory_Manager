ProcessorUtils:SetWinningVal_ByCompareResult (baseValue, challengerValue, comparator, winnersTable, targetPartyMember)
Calculates the winner based on the result of comparing the baseValue against the challengerValue, using the Comparator to determine the nature of the compare

Parameters:

  • baseValue number|nil
  • challengerValue number
  • comparator CompareStrategy
  • winnersTable table
  • targetPartyMember GUIDSTRING

Returns:

  1. table of winners - will either append the targetPartyMember if the values were equal, or replace the table with just targetPartyMember if the challenger won
  2. number that won in the compare, or baseValue if both were equal
ProcessorUtils:CalculateTotalItemCount (targetsWithAmountWon, targetChar, inventoryHolder, root, item)
Uses the following on the targetChar + Osi.GetStackAmount (via Osi.GetItemByTemplateInInventory) + the calculated amount won for this item stack thusfar + the calculated amount won for previous items of the same template that haven't been added to the targetChar inventory yet (event hasn't been processed) to determine the amount of the item's template that are theoretically in the given characters inventory. If the targetChar is the inventoryHolder, will subtract the amount of the item stack being processed that has been "won" by the other party members

Parameters:

  • targetsWithAmountWon table
  • targetChar CHARACTER
  • inventoryHolder CHARACTER
  • root GUIDSTRING
  • item GUIDSTRING

Returns:

    number the calculated stack size
