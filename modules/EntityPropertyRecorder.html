BuildInitialRecordEntry (initialApplicableItemFilterMaps, initialApplicableItemFilterFields, initialApplicableFilterFields, initialApplicablePreFilterFields, initialValue)
Constructs the common table structure shared by all recorded entries

Parameters:

  • initialApplicableItemFilterMaps will default to {} if not provided
  • initialApplicableItemFilterFields will default to {} if not provided
  • initialApplicableFilterFields will be removed if nil is provided
  • initialApplicablePreFilterFields will be removed if nil is provided
  • initialValue will default to "N/A" if not provided

Returns:

    table
RegisterPropertyRecorders (modUUID, ...)
Registers new Entity Property Recorders for the given mod

Parameters:

  • modUUID that ScriptExtender has registered for your mod, for tracking purposes - https://github.com/Norbyte/bg3se/blob/main/Docs/API.md#ismodloadedmodguid will throw an error if the mod identified by that UUID is not loaded
  • ... function the functions to register. Each function should accept the following params:
    GUIDSTRING - of the entity being proceseed. Will be the character string or the item UUID, but not the rootTemplateUUID (unless a mod manually triggers recording on one)
    and return the record entry as a table to save to the file, or nil. Can be any structure, but recommended to use the existing one (can use EntityPropertyRecorder:BuildInitialRecordEntry to construct it)

Returns:

    boolean true if the operation succeeds
RecordEntityProperties (entity, respectAimDisabledConfig)
Runs all the registered recorders against the entity and saves them to the file, if the RECORD_APPLICABLE_ENTITY_PROPS config is enabled Each recorder is run inside of pcall, so all errors will be logged, but not propagated. Runs an Osi.Exists check first, logging a warning if it returns false Automatically run whenever an item without the AIM_PROCESSED tag is being run through AIM, or after a character joins the party

Parameters:

  • entity string the character GUIDSTRING or the item GUIDSTRING (not the rootTemplate)
  • respectAimDisabledConfig boolean true to respect the ENABLED AIM config property, false to only respect the RECORD_APPLICABLE_ENTITY_PROPS config. Defaults to true

Returns:

    table the recorded result for the entity, or nil if processing wasn't enabled or the entity doesn't exist
RecordPartyMembers (respectAimDisabledConfig)
Executes EntityPropertyRecorder:RecordEntityProperties against the current party members (as found by Osi.DB_Players:Get(nil)) Automatically run after the `Osi.CharacterJoinedParty` and `LevelGamplayStarted` event

Parameters:

  • respectAimDisabledConfig boolean true to respect the ENABLED AIM config property, false to only respect the RECORD_APPLICABLE_ENTITY_PROPS config. Defaults to true
